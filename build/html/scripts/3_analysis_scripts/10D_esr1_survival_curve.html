GEPIA2 Run ThroughÔÉÅ

This script demonstrates a programmatic way to generate Kaplan-Meier survival curves using the GEPIA2 (Gene Expression Profiling Interactive Analysis) database. It specifically focuses on analyzing how the expression of certain genetic transcripts correlates with patient survival outcomes in breast cancer.
