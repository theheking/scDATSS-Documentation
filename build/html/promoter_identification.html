Overview

This section contains three main analyses for identifying and characterizing alternative promoters:

  • 1A: Integration Analysis - Combines PacBio long-read data, CAGE-seq TSS data, and proActiv predictions

  • 1B: Target Region Identification - Defines optimal regions for dual guide targeting

  • 1C: RefTSS Comparison - Validates identified promoters against reference annotations
